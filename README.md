~ Fashion Lookbook Website

A modern, minimal, and elegant **Fashion Lookbook Website** built using **HTML and CSS** (with CSS Grid & Flexbox).  
It showcases seasonal trends, designer highlights, and a curated fashion gallery.


~ Features

- **Hero Section** — Stylish banner highlighting the latest collection.  
- **Gallery Grid** — Responsive photo grid for curated fashion looks.  
- **Trends Section** — Displays current seasonal fashion insights.  
- **Designers Section** — Highlights top designers with images and bios.  
- **About Section** — Story behind the digital lookbook.  
- **Newsletter** — Subscription box to join the fashion digest.  
- **Responsive Layout** — Fully optimized for desktop and mobile screens.


~ Technologies Used

- **HTML5** — Semantic structure for clean and accessible markup.  
- **CSS3** — Used Flexbox and Grid for layout, hover effects, and responsive design.  
- **Google Fonts / Serif Fonts** — For elegant typography.  
- **Pinimg Image Assets** — For placeholder lookbook photos.
